# RE-DACT
RE-DACT: Secure Redaction/Anonymization Tool
It is an ML-powered document redaction and anonymization system that automatically detects and masks sensitive information such as names, phone numbers, email addresses, and other personally identifiable information (PII) from PDFs, images, receipts, Excel sheets, and text documents using OCR and intelligent text extraction. It provides multiple levels of redaction, including masking, full hiding, and synthetic data replacement, to ensure privacy, secure data sharing, and compliance across real-world business use cases.
