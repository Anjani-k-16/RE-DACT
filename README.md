# RE-DACT

### ML-Powered Secure Document Redaction & Anonymization System

RE-DACT is an **ML-powered document redaction and anonymization platform** designed to automatically detect, mask, and protect sensitive personally identifiable information (PII) from multiple document formats.

The system intelligently processes **PDFs, images, receipts, text files, and spreadsheets** using **OCR and machine learning–based text extraction** to identify confidential information such as:

* Names
* Email addresses
* Phone numbers
* Locations
* Organization names
* Other sensitive entities

RE-DACT provides **multiple levels of privacy protection**, including masking, full redaction, and synthetic data replacement, making it suitable for **secure data sharing, privacy compliance, and enterprise document workflows**.

##  Key Features

*  **Automatic PII Detection**
  Identifies sensitive information from structured and unstructured documents.

*  **Multi-Format Support**
  Supports PDFs, scanned images, receipts, text files, and Excel sheets.

*  **ML + OCR Powered Extraction**
  Uses intelligent OCR scanning and machine learning techniques for accurate entity extraction.

*  **Multi-Level Redaction**
  Offers configurable redaction levels:

  * Partial masking
  * Full redaction
  * Advanced anonymization
  * Synthetic data replacement

*  **User-Friendly Web Interface**
  Built with Streamlit for simple and interactive document processing.

*  **Real-World Enterprise Use Cases**
  Useful in:

  * Banking
  * Insurance
  * Healthcare
  * Legal documentation
  * Compliance workflows

##  Problem Statement

Organizations often handle documents containing highly sensitive personal information. Manual redaction is time-consuming, error-prone, and not scalable.

RE-DACT solves this problem by automating the process of **detecting and securely anonymizing sensitive information**, reducing privacy risks and improving compliance efficiency.
